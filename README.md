# Web3Auth Core x Facebook Login Demo Application

## Quick Start

```bash
git clone https://github.com/shahbaz17/web3auth-core-facebook-demo.git
cd web3auth-core-facebook-demo
```

## Installation

Run:

```bash
npm install
```

## Start

Run:

```bash
npm run start
```

## Configure

Follow https://web3auth.io/docs/custom-authentication/social-providers/facebook
to create a Facebook application and set up Facebook Custom Authentication.

## Guide

https://web3auth.io/docs/guides/facebook
